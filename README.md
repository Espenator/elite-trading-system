# 🎯 Elite Trading System

A quantitative momentum trading system built with Python, featuring advanced technical analysis, multi-factor scoring, and real-time market scanning.

## 🚀 Features

- **Bible Score Engine**: 50+ technical indicators weighted for momentum trading
- **Structure Analysis**: Candlestick patterns, support/resistance, and trend analysis
- **Real-time Scanning**: Finviz integration for market-wide opportunity detection
- **Professional Dashboard**: Streamlit-based UI with beautiful signal cards
- **Risk Management**: Automatic stop-loss and target calculation based on ATR
- **Multi-regime Detection**: Adaptive strategies for GREEN/YELLOW/RED market conditions
