# Bull Market Directives

## Entry Rules
- Favor momentum-following entries
- Buy on pullbacks to SMA20 with volume confirmation
- Wider stops: use ATR x 2.0 for stop placement
- Scale into positions on confirmed breakouts

## Position Sizing
- Position scale multiplier: 1.2x normal
- Allow up to 8% portfolio heat in strong trends
- Increase max single position to 3%

## Exit Rules
- Use trailing stops rather than fixed targets
- Let winners run — raise stops on momentum continuation
- Take partial profits at 2R, let remainder ride

## Bias
- Default bias: LONG
- Short only on clear technical breakdowns with volume
- Momentum > mean-reversion in bull regimes
